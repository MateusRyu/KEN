## Goal

The main goal of this project is to consolidate the knowledge gained in the back-end Java specialist training proposed by
Alura and Oracle. This is a Spring Java application that allows users to search books and authors with a external API and 
a database. The main skills tested in this challenge were:

- Use Spring framework;
- Store data on database;
- Get data from database;
- Security of sensible data.
